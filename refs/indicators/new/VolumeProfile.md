# VP (Volume Profile)

성격: 수급/분포 지표(기타)
권장기간: 중기~장기(프로필은 세션·주간·월간 단위로 나눠서 해석)
시그널: - POC(Point of Control, 거래량 집중 구간) = 강한 지지/저항. 

- VAH/VOL(Volume Area High/Low)로 매수·매도 구역 판단.
주의: 시야(세션 길이)에 따라 전혀 다른 분포가 나옴 → 항상 동일한 기간 기준으로 비교.
정리된 블럭: Volume Profile(POC) (https://www.notion.so/Volume-Profile-POC-29825237772c80449d9ac917c907f8a2?pvs=21) 
설명: 특정 기간(세션/일/주)의 가격 축(가격 레인지)에 체결된 거래량을 수평 막대로 표시.

지지/저항 식별에 매우 유용

## **📘 Volume Profile (VP, 거래량 프로파일)**

> “가격대별 거래량 집중 구간을 시각화해, 시장의 진짜 지지·저항을 찾아내는 도구.”
> 

> 가격의 ‘시간’이 아닌,
> 
> 
> **거래량이 쌓인 가격 구간**
> 

> 현대적
> 
> 
> **수평형 시장 구조 지표(Horizontal Volume Distribution Indicator)**
> 

---

### **① 개념 및 계산식**

| **항목** | **내용** |
| --- | --- |
| **정의** | Volume Profile은 일정 구간 동안 **각 가격대별로 거래된 총 거래량을 수평 히스토그램** 형태로 표시한 지표입니다. 즉, “어느 가격에서 거래가 가장 많이 발생했는가?”를 통해 **시장 참여자들의 평균 체결가 분포**를 보여줍니다. |
| **공식(개념적)** | 거래량 V_i를 가격 구간 P_i별로 누적하여 계산:   **VP(P) = Σ 거래량(V) / 각 가격 구간별 (ΔP)** |
| **핵심 구성요소** | - **POC (Point of Control):** 거래량이 가장 많은 가격대 (시장 균형 중심) - **VAH (Value Area High):** 전체 거래량의 상위 70% 중 가장 높은 가격대 - **VAL (Value Area Low):** 전체 거래량의 상위 70% 중 가장 낮은 가격대 |
| **핵심 개념 요약** | 가격의 “시간” 대신 “거래량의 밀도”로 시장의 **균형·불균형 상태**를 측정. |

> 💡
> 
> 
> **한마디로:**
> 

> “Volume Profile은 ‘어디서 얼마만큼 거래가 쌓였는가’를 보여주는 실질적 매물대 지도입니다.”
>

### **② 분류**

| **구분** | **내용** |
| --- | --- |
| **지표 유형** | **거래량 기반형 (Volume-based Structural Indicator)** |
| **분류 체계상 위치** | 추세형도 오실레이터형도 아님 → **시장구조형(Market Structure Tool)** |
| **주요 목적** | 지지·저항대 식별, 세력 매집 구간 포착, 밸류존(Value Area) 분석 |
| **사용자층** | 퀀트 트레이더, 기관 포지션 트레이더, 프로 데이 트레이더 |

---

### **③ 신호 해석 (매매 시그널)**

| **구분** | **해석** | **실전 전략** |
| --- | --- | --- |
| **POC (Point of Control)** | 거래량이 가장 많은 가격대 → “시장 밸런스 중심” | 가격이 POC 위 → 강세 / 아래 → 약세 경향 |
| **VAL~VAH (Value Area)** | 전체 거래량의 약 70%가 형성된 구간 = 공정가치 영역 | 가격이 Value Area 안 → 횡보 / 바깥 → 추세 이동 가능성 |
| **VAL 하단 이탈 후 복귀** | 밸류존 이탈 후 재진입 → **매수 신호 (수급 복귀)** | 매수 진입 유리 구간 |
| **VAH 상단 돌파 + 거래량 증가** | 거래량 동반 상향 돌파 → **강세 지속 신호** | 매수 추세 강화 |
| **POC 하향 이탈 + 거래량 확산** | 세력 이탈 or 분산 발생 → **약세 전환 가능성** | 매도 신호 or 숏 포지션 진입 |
| **Thin Volume Zone(빈 거래대 구간)** | 거래량이 적은 가격 구간 → **가격이 빠르게 이동하는 통로** | 갭 이동 / 급등락 발생 가능 구간 |

> 📊
> 
> 
> **예시:**
> 

> 테슬라(TSLA) 주가가 $250~$255 구간에서 거래량 집중 (POC=$252).
> 

> 가격이 $252 아래로 내려가면 매도세 확산, $255 돌파 시 강세 확정 패턴.
>

### **④ 권장 기간 및 적용 타임프레임**

| **기간** | **활용도** | **사용 예시** |
| --- | --- | --- |
| **단기 (1분~15분봉)** | Intraday Volume Profile → 당일 매물대 및 스캘핑용 | 데이 트레이딩, 단기 눌림 진입 판단 |
| **중기 (1시간~일봉)** | 스윙 매매용 핵심 구간 → 주요 수급 균형 구간 확인 | 스윙 트레이딩, 분할 매수/매도 구간 설정 |
| **장기 (주봉~월봉)** | 장기 밸류존 및 세력 매집대 확인 | 포지션 트레이딩, 대세 구간 판단 |

> 💡
> 
> 
> **실전 팁:**
> 

> 대부분 트레이더는 “Composite VP (전체 구간)”과 “Session VP (일별 구간)”을 함께 사용.
> 

> → 장기 밸류존과 단기 밸류존이 겹치는 구간 =
> 
> 
> **핵심 매물 집중대 (고신뢰 지지선)**
>

### **⑤ 주의사항 및 실전 팁**

| **항목** | **설명** |
| --- | --- |
| **주의1** | 거래량이 적은 종목은 신뢰도가 낮음 (노이즈 발생). |
| **주의2** | 시세 급등/급락 후에는 새로운 VP를 갱신해야 함 (이전 VP 무의미). |
| **주의3** | VAH/VAL 돌파 시 거래량 확인 필수 — 단순 돌파보다 수급이 중요. |
| **활용 팁1** | **VP + VWAP** → 평균 단가와 매물대 중심을 함께 파악 가능. |
| **활용 팁2** | **VP + Anchored VWAP** → 특정 이벤트 이후 세력 매집 구간 정밀 분석. |
| **활용 팁3** | **VP + RSI/Stoch** → 밸류존 이탈 후 과매도 확인 시 반등 타이밍 포착. |
| **활용 팁4** | **POC 이동 방향** 자체가 수급 중심 이동을 의미 → 중기 추세 판단 가능. |

### **⑥ 종합 요약 및 실전 응용 예시**

| **항목** | **내용** |
| --- | --- |
| **핵심 요약** | Volume Profile은 시장 참여자들이 **어디서 가장 많이 거래했는가**를 시각적으로 보여주는 도구로, **지지/저항, 밸류존, 세력 매집 구간**을 정밀하게 보여줍니다. |
| **실전 응용 예시** | - 주가가 VAH 상단 돌파 시 거래량 동반 → **추세 돌파 확인 매수** - VAH 돌파 실패 후 POC 아래 이탈 → **단기 매도 신호** - VAL 근처 거래량 급증 → **수급 복귀형 반등 가능 구간** |
| **추천 조합** | Volume Profile + Anchored VWAP + EMA(50) |
| **현대적 활용** | 기관·퀀트 트레이더들은 VP를 **시장 미시구조(Market Microstructure)** 분석의 핵심으로 사용. 특히 NASDAQ/NYSE 데이터 기반의 **High-Volume Nodes(HVN)**, **Low-Volume Nodes(LVN)** 분석을 통해 “세력의 매집/분산 구간”을 정량화함. |
