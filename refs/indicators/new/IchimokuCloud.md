# IC (일목균형표, Ichimoku Cloud)

성격: 추세 지표
권장기간: 중기~장기 (단기에도 쓰지만, 일목의 원래 목적은 중~장기 추세판단)
시그널: - 구름 위/아래에 위치 여부(추세 상태), 전환선↔기준선 교차(단기 전환), 구름 돌파(중·장기 전환 신뢰도 높음).
권장 파라미터: 표준 (9,26,52)
주의: 복합적이므로 각 요소(전환선, 기준선, 구름, 후행스팬)를 분리해 해석해야 함.
정리된 블럭: 일목균형표 (https://www.notion.so/29825237772c80409bcddbe4814491e3?pvs=21) 
설명: 전환선(9), 기준선(26), 선행스팬(선행A, 선행B=52) 등 여러 구성 요소로 추세·지지/저항·균형을 한 번에 제공. 

후행스팬은 과거 가격과의 관계로 추세 확인.

추세 방향·구름(지지/저항)·전환 신호(구름 돌파, 기준선 교차) 제공.

## **☁️ IC (Ichimoku Cloud, 일목균형표)**

| **구분** | **내용** |
| --- | --- |
| **지표 유형** | 추세형 + 복합형 (Trend + Momentum + Support/Resistance) |
| **창시자** | 고이치 호사다 (Goichi Hosoda, 일본 기자, 1930년대 개발, 1969년 공개) |
| **영문명** | Ichimoku Kinko Hyo (一目均衡表) = “한눈에 균형을 본다” |
| **핵심 목적** | 가격의 **균형 상태(Equilibrium)**와 **추세 방향**을 동시에 시각화하여, “한눈에 시장의 흐름을 파악” |
| **구성 요소** | 전환선(Tenkan-sen), 기준선(Kijun-sen), 선행스팬1·2(Senkou Span A/B), 후행스팬(Chikou Span) — 총 5가지 |

## **① 개념 및 공식 정의**

| **구성요소** | **수식** | **의미** | **해석 포인트** |
| --- | --- | --- | --- |
| **전환선 (Tenkan-sen)** | (최근 9일 고가 + 9일 저가) ÷ 2 | 단기 추세선 | 단기 모멘텀, 방향 전환 신호 |
| **기준선 (Kijun-sen)** | (최근 26일 고가 + 26일 저가) ÷ 2 | 중기 추세선 | 추세의 기준선, 지지/저항 역할 |
| **선행스팬 1 (Senkou Span A)** | (전환선 + 기준선) ÷ 2 (26일 선행 표시) | 단기·중기 평균선 | 구름대 상단/하단의 빠른 경계 |
| **선행스팬 2 (Senkou Span B)** | (최근 52일 고가 + 52일 저가) ÷ 2 (26일 선행 표시) | 장기 추세선 | 구름대의 완만한 경계선 |
| **후행스팬 (Chikou Span)** | 현재 종가를 26일 뒤로 이동 | 시간적 확인선 | 가격과의 괴리로 추세 확인 |

⸻

👉 **구름(Cloud, Kumo)** = Senkou Span A & B로 형성된 영역.

- Span A > Span B → 구름이 상승형(강세)
- Span A < Span B → 구름이 하락형(약세)

## **② 지표 성격 (분류 및 본질)**

| **항목** | **내용** |
| --- | --- |
| **지표 분류** | **추세형 복합 지표 (Trend-Following + Momentum)** |
| **기능 분류** | 추세 방향, 지지·저항, 타이밍, 모멘텀 동시 파악 |
| **시장 반응형 분류** | 후행적이지만 **예측형 구간(구름)** 포함 |
| **핵심 강점** | 지표 하나로 “현재 추세 + 미래 균형 구간 + 타이밍”을 동시에 시각화 |
| **유형적 특성** | 이동평균보다 완화되고, MACD보다 부드러우며, Bollinger보다 구조적 |

## **③ 신호 해석 (실전 활용 기준)**

| **구분** | **조건** | **해석** | **의미** |
| --- | --- | --- | --- |
| **기본 추세 판단** | 종가가 구름 위 | 상승 추세 | 매수 관점 |
|  | 종가가 구름 아래 | 하락 추세 | 매도 관점 |
|  | 종가가 구름 내부 | 중립, 조정 구간 | 관망 |
| **전환선 vs 기준선** | 전환선이 기준선을 상향 돌파 | **골든크로스 (매수 신호)** | 단기 모멘텀 상승 |
|  | 전환선이 기준선을 하향 돌파 | **데드크로스 (매도 신호)** | 단기 모멘텀 약화 |
| **후행스팬** | 후행스팬이 과거 캔들 위 | 상승 추세 확정 | 추세 지속 신호 |
|  | 후행스팬이 과거 캔들 아래 | 하락 추세 확정 | 약세 지속 신호 |
| **구름 돌파 (Kumo Breakout)** | 종가가 구름대를 상향 돌파 | 강한 매수 신호 | 추세 전환 가능성↑ |
|  | 종가가 구름대를 하향 돌파 | 강한 매도 신호 | 추세 반전 가능성↑ |

📌 **핵심 포인트:**

단기 신호(전환/기준선 교차)와 중기 신호(구름 돌파), 장기 신호(후행스팬 위치)가 **3단계로 일치할 때** 가장 강력한 확증 신호가 됩니다.

## **⑤ 주의점 및 실전 팁**

| **항목** | **내용** |
| --- | --- |
| **1. 횡보장 약점** | 구름대 안에서는 신호 신뢰도 ↓ (가격이 구름 내부에서 흔들림) |
| **2. 복합 확인 필수** | IC 단독보다 **MA·ADX·Volume Profile** 병행 시 추세 강도 판단↑ |
| **3. 시간 요소 주목** | 일목균형표의 핵심은 “가격뿐 아니라 시간” → 구름대 두께와 위치로 **추세 피로도 및 전환 시점 예측** |
| **4. 구름대 두께 해석** | 두꺼운 구름 = 강한 지지/저항, 얇은 구름 = 돌파 가능성 높음 |
| **5. 다중 타임프레임** | 상위 타임프레임(4H, 1D)의 구름방향이 하위 타임프레임 추세를 지배함 |
| **6. 후행스팬 역할** | 과거 가격과의 괴리로 “추세 확정 신호” 제공 (확증용 필터로 유용) |

## **⑥ IC 종합 요약표**

| **항목** | **내용** |
| --- | --- |
| **분류** | 추세형 복합 지표 (Trend + Momentum + Equilibrium) |
| **주 목적** | 시장의 균형 상태(가격·시간) 시각화, 추세 전환 및 지속 구간 파악 |
| **핵심 요소** | 전환선·기준선·구름대·후행스팬 |
| **권장 기간** | (9,26,52) 표준 / 단기(7,22,44) / 장기(20,60,120) |
| **적합 시장** | 추세장, 중기 이상 파동 |
| **비적합 시장** | 횡보장, 저변동성 구간 |
| **추천 조합** | IC + ADX (추세 강도 확인), IC + RSI (모멘텀 확인), IC + VWAP (실제 체결 밸런스 확인) |
| **장점** | 추세·지지선·저항선·타이밍을 동시에 제공 |
| **단점** | 단기 신호 많음, 횡보장 오신호 발생 |

## **🔧 실전 응용 예시**

| **전략명** | **조건** | **의미** |
| --- | --- | --- |
| **Kumo Breakout 전략** | 가격이 구름 위 돌파 + 전환선>기준선 + 후행스팬>캔들 | 상승추세 확정 → 매수 |
| **Bearish Reversal 전략** | 가격이 구름 하향 돌파 + 전환선<기준선 + 후행스팬<캔들 | 하락추세 확정 → 매도 |
| **Sideways 필터 전략** | 가격이 구름 내부 체류 | 방향성 약함 → 진입 회피 |
| **Re-entry 전략** | 구름 돌파 후 가격이 구름 상단 재테스트 후 반등 | 구름이 지지로 전환 → 재진입 기회 |
