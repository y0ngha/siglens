# VWAP (Volume Weighted Average Price)

성격: 추세 지표
권장기간: 단기(데이 트레이드 / 당일) — 일봉 이상에선 쓰임새 제한적.
시그널: - 가격 > VWAP → 매수 우위(기관 바이인), 가격 < VWAP → 매도 우위. 

- VWAP 리테스트(지지/저항)에서 롱/숏 판단.
권장 파라미터: Bands: 1, 1.5, 2
주의: 단기 트레이딩용으로 쓸 것.
정리된 블럭: VWAP (https://www.notion.so/VWAP-29825237772c801c842ce569288761db?pvs=21) 
설명: 당일 거래량으로 가중한 평균가격. 세션(당일) 단위의 가격 벤치마크.

## **📘 VWAP (Volume Weighted Average Price)**

> “거래량이 반영된 공정가치(price of fairness)”
> 

> → 기관의 매수·매도 평균단가이자, 단기 지지/저항 및 세력 매집 여부 판단 지표
> 

---

### **① 개념 및 계산식**

| **항목** | **내용** |
| --- | --- |
| **정의** | VWAP는 거래량을 가중치로 하여 계산된 평균 가격으로, 특정 기간 동안의 “실질 평균 매매단가”를 의미함. 기관이나 펀드가 대량 매매할 때 “공정가격”으로 사용. |
| **공식** | **VWAP = (Σ (거래가격 × 거래량)) ÷ (Σ 거래량)** |
| **핵심 해석** | 단순 평균가(MA)와 달리, 거래량이 많을수록 해당 가격의 비중이 커짐 → “실제 시장 참여자들의 체결중심가”로서 신뢰도가 높음. |
| **특징** | ① 실시간으로 갱신됨 (Intraday 중심) ② 거래량 급증 시 급격히 변동 ③ 일간 단위로 초기화 (보통 장 시작 기준으로 재계산) |

---

### **② 분류**

| **구분** | **내용** |
| --- | --- |
| **지표 유형** | **추세형 + 거래량 기반 지표 (Hybrid)** |
| **관점** | 주로 단기 및 당일 매매용 (Day Trading / Scalp Trading) |
| **주요 사용자** | 기관, 알고리즘 트레이더, 시장 메이커, 단타 트레이더 |

---

### **③ 신호 해석 (매매 시그널)**

| **구분** | **해석** | **매매전략** |
| --- | --- | --- |
| **가격 > VWAP** | 매수세 우위. 시장이 VWAP 위에서 유지될 경우 “세력의 평균 매수단가 위에 있음” → **상승 추세 유지** | **매수/롱 포지션 유리** |
| **가격 < VWAP** | 매도세 우위. 평균 매입가보다 낮은 수준에서 거래 중 → **하락 압력** | **매도/숏 포지션 유리** |
| **VWAP 돌파 (상향)** | 거래량 동반 시 → 강한 추세 전환 (단기 돌파 시점) | **단타 매수 진입 시그널** |
| **VWAP 이탈 (하향)** | 거래량 동반 시 → 매도세 강화 | **단타 매도 진입 시그널** |
| **VWAP 근처 횡보** | 관망 구간, 세력 매집 가능성 | **브레이크아웃 대기** |

> 💡
> 
> 
> **Tip:**
> 

> VWAP 아래 매집 → 기관 매수 유입 신호 / VWAP 위 분할 매도 → 차익실현 신호.
>

### **④ 권장 기간(적용 타임프레임)**

| **기간** | **활용도** | **세부 설정** |
| --- | --- | --- |
| **단기 (1~15분봉)** | ✅ 매우 중요 — 단타 트레이더용 당일 흐름, 진입·청산 기준 | 매수: 가격이 VWAP 상향 돌파 시 청산: VWAP 이탈 시 |
| **중기 (1시간~4시간봉)** | ⚪ 참고용 — 추세 전환 확인 | 기관 매집 구간 식별 |
| **장기 (일봉 이상)** | ⚪ 사용 빈도 낮음 → 대신 **Anchored VWAP (고정 기준점 VWAP)** 사용 | 특정 이벤트(예: 실적발표일, 갭 발생일) 기준으로 설정 |

### **⑤ 주의사항 및 활용 팁**

| **구분** | **설명** |
| --- | --- |
| **주의1** | VWAP은 **거래량이 적은 종목**에서는 왜곡될 수 있음 (허수체결 영향). |
| **주의2** | **장중 초기(첫 15분)** 데이터는 변동성이 매우 크므로 신호 신뢰도 낮음. |
| **주의3** | 일봉 이상에서는 **리셋되지 않는 Anchored VWAP**을 병행해야 정확. |
| **활용 팁** | - VWAP + 볼린저밴드 → “Volume-Weighted Bollinger”로 변동성+거래량 결합 - VWAP + RSI → 단타 과열 진입·이탈 판단 - VWAP + Volume Profile → 세력 매집 구간 정밀 확인 |

---

### **⑥ 요약 및 실전 예시**

| **구분** | **내용** |
| --- | --- |
| **핵심 요약** | VWAP은 거래량을 반영한 공정가격으로, **기관의 평균 매수단가 및 시장심리의 중심축** 역할을 함. 단타 매매의 기준선으로 매우 강력한 신뢰도. |
| **실전 예시** | 예: TSLA가 오전 10시 VWAP 255달러 돌파 + 거래량 급증 → 단타 매수 진입. 오후에 VWAP 재이탈 시 청산 → **5% 수익 확보 사례**. |
| **추천 조합** | VWAP + MA(20) + Volume Profile = 세력 매집 구간 + 지지선 확인 |
| **현대적 활용** | 2020년 이후 AI/Algo 트레이더들이 VWAP을 **Execution Benchmark**로 사용하여 대량주문의 체결 효율을 평가함. (특히 Morgan Stanley, Citadel 등 기관 트레이딩 시스템에서 핵심 지표로 사용됨.) |
