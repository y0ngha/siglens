# ADX (Average Directional Index)

성격: 추세 강도 지표, 추세 지표
권장기간: 중기~장기 (일봉 ADX(14) 기본).
시그널: - ADX 상승 +DI/−DI 구분으로 ‘강한 추세의 방향’ 판단. 

- ADX가 낮으면 레인지(오실레이터 신호 활용).
정리된 블럭: ADX(Average Directional Movement Index) (https://www.notion.so/ADX-Average-Directional-Movement-Index-29a25237772c80baa6acff79b6874b84?pvs=21) 
설명: DMI 기반으로 추세의 ‘강도’만을 수치화(0~100). 보통 25 이상이면 의미 있는 추세.

## **📘ADX (Average Directional Index)**

> “현재 시장에 추세가 존재하는가, 그리고 그 추세가 얼마나 강한가를 측정하는 지표.”
> 

ADX는 **DMI의 중심축**이자, 추세의 **“방향이 아닌 강도”**만을 측정합니다.

즉, 시장이 상승이든 하락이든 **“추세의 힘”**만 보는 지표입니다.

---

### **① 개념 및 계산식**

| **항목** | **내용** |
| --- | --- |
| **정의** | ADX는 +DI(상승압력)와 −DI(하락압력)의 차이를 기반으로 **현재 추세 강도(Trend Strength)**를 0~100 사이의 값으로 표현합니다. |
| **핵심 역할** | 방향(상승/하락)을 판단하지 않음 — 오직 **추세의 존재 여부**와 **강도**만 측정 |

| **표준 설정** | 14기간 (RSI와 동일) |
| **값의 의미** | - 020 → 추세 거의 없음 (횡보장)- 2025 → 추세 형성 시작- 25~40 → 추세 강화 구간- 40 이상 → 매우 강한 추세 (모멘텀 트레이딩 유리) |

> 💡
> 
> 
> **핵심 포인트:**
> 

> ADX는 +DI/−DI가 어느 쪽이든 상관없이,
> 

> 추세가 강해지면 ADX가 올라가고, 추세가 약해지면 ADX가 내려갑니다.
>

### **② 지표 성격 (분류)**

| **항목** | **내용** |
| --- | --- |
| **분류** | **추세형 지표 (Trend Strength Indicator)** |
| **범위** | 0 ~ 100 (일반적으로 20 이상이면 추세 존재로 간주) |
| **중심 역할** | DMI의 핵심 지표 — 추세 강도 판단 |
| **장점** | - 시장에 추세가 존재하는지 명확히 구분 가능- 상승/하락 모두에서 활용 가능 |
| **단점** | - 방향성은 알려주지 않음 (+DI/−DI 필요)- 추세 초반에는 신호 지연 발생 가능 |

---

### **③ 실전 해석 (시그널 및 활용법)**

| **ADX 수치** | **추세 강도 해석** | **실전 매매 전략** |
| --- | --- | --- |
| **0 ~ 20** | 추세 없음 (횡보장) | 박스권 매매 / 관망 |
| **20 ~ 25** | 추세 형성 초기 | 추세 시작 탐지 구간 (매수·매도 준비) |
| **25 ~ 40** | 뚜렷한 추세 진행 | 추세 추종 매매 유리 (트렌드 팔로잉 구간) |
| **40 ~ 60** | 강한 추세 지속 | 모멘텀 매매 강화, 반전 경계 |
| **60 이상** | 과열 구간 | 단기 피로 누적 — 익절 또는 분할 청산 고려 |
| **ADX 상승 중** | 추세 강화 중 | 현재 방향 유지 (추세 지속 가능성 높음) |
| **ADX 하락 중** | 추세 약화 중 | 추세 종료 가능성, 진입 자제 |

> ⚙️
> 
> 
> **응용 예시:**
> 
- +DI > −DI & ADX 상승 → 상승 추세 강화 (매수 지속)
- −DI > +DI & ADX 상승 → 하락 추세 강화 (매도 지속)
- ADX 하락 → 추세 약화 (익절 or 관망 신호)

### **④ 권장 기간 및 타임프레임**

| **구분** | **권장 설정** | **해석 기준** |
| --- | --- | --- |
| **단기 (5~10)** | 변동성 높은 단기 트레이딩용 | 빠른 추세 전환 감지 |
| **중기 (14)** | 표준값 — 스윙 및 데일리 트레이딩용 | 가장 많이 사용되는 기본 설정 |
| **장기 (20~30)** | 장기 추세 필터링용 | 방향성 큰 추세만 포착 |

> 💡
> 
> 
> **추세형 조합 예시:**
> 
- ADX(14) + EMA(50) : 중기 추세 확인
- ADX(14) + MACD : 추세 강도 + 모멘텀 일치 판단
- ADX(20) + RSI(14) : 추세 지속 + 과매수/과매도 필터링

### **⑤ 주의사항 및 실전 팁**

| **구분** | **설명** |
| --- | --- |
| **주의 1** | ADX는 방향이 아닌 “강도”만 의미하므로 반드시 **+DI / −DI**와 함께 해석해야 함 |
| **주의 2** | ADX가 하락한다고 해서 역추세 진입 금지 — 단순히 “추세 약화” 의미 |
| **주의 3** | 초기 상승 신호(20 돌파)는 가짜 신호일 수 있음 — 이평선 돌파 확인 병행 |
| **활용 팁 1** | ADX 상승 구간에서 **EMA 방향과 일치**할 때 진입 신뢰도 상승 |
| **활용 팁 2** | ADX 20 → 25 돌파는 “추세 시작” 신호로 매수·매도 관찰 포인트 |
| **활용 팁 3** | ADX 피크 이후 하락 전환 → “추세 피로”로 익절 타이밍 판단 가능 |
