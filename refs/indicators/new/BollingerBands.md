# BB (Bollinger Bands)

성격: 변동성 측정 지표, 추세 지표
권장기간: 단기~중기 (20-period 기준이 가장 보편적).
시그널: - 밴드 수축 → 돌파(확장) 시 추세 시작 가능성(돌파 방향으로).

- 가격이 상단에 지속 머무르면 추세(상승) 지속 가능성. 

- 밴드 터치 자체는 약한 신호(특히 강추세에서는 무시되기도 함).
권장 파라미터: 기본 (20, 2). 단기선(10,2)으로 민감도 ↑, 스윙용은 (20,2) 유지.
주의: 변동성·추세 확인을 위해 볼린저 단독 사용 금지 — 거래량·추세지표(EMA, ADX)와 동시 확인 필요.
정리된 블럭: 차트는 답을 알고있다 | 존 볼린저 John Bollinger | 토스증권 INVESTORS 25 (https://www.notion.so/John-Bollinger-INVESTORS-25-28b25237772c80388281f7064661fa01?pvs=21) 
설명: 중앙선 = 보통 20기간 MA, 상/하밴드 = 평균 ± k·표준편차(보통 k=2).

## **🔶 Bollinger Bands (볼린저 밴드)**

---

### **① 개념 및 수식 정의**

| **항목** | **내용** |
| --- | --- |
| **창시자** | John Bollinger (1983년) |
| **핵심 개념** | 변동성(Volatility)을 이용해 주가의 상대적 고점·저점을 시각화하는 **표준편차 기반 밴드형 지표** |
| **기본 구성** | 중심선(Middle Band), 상단선(Upper Band), 하단선(Lower Band) |
| **기본 수식** | **Middle Band** = n기간 단순이동평균(SMA)**Upper Band** = SMA + (k × 표준편차)**Lower Band** = SMA − (k × 표준편차) |
| **기본 설정값** | n = 20, k = 2 (즉, 20일 이동평균 ± 2σ) |
| **핵심 아이디어** | 시장이 조용할수록 밴드 폭이 좁아지고, 변동성이 커질수록 밴드 폭이 넓어짐 → 밴드 수축은 **변동성 폭발(추세 시작)**의 전조 신호로 활용 가능 |

📌 **요약:**

볼린저 밴드는 가격의 평균과 표준편차를 동시에 반영하여,

- *“가격이 통계적으로 평균에서 얼마나 벗어났는가”**를 보여주는 **변동성 지표 + 추세 보조지표**입니다.

---

### **② 지표 성격 (분류 및 특성)**

| **항목** | **내용** |
| --- | --- |
| **분류** | 복합형 (변동성 기반 추세/오실레이터 성격 모두 포함) |
| **핵심 목적** | 가격의 상대적 위치(평균 대비 과열/과냉) 및 변동성 확장·수축 확인 |
| **지표 유형 비교** | 단순 오실레이터(RSI, CCI)처럼 ‘과매수·과매도’를 직접 계산하지 않지만, 밴드의 경계로 과열을 시각화함 |
| **대표 해석 방식** | 밴드 상단 접촉 = 과매수 가능성 / 밴드 하단 접촉 = 과매도 가능성 / 밴드 수축 후 돌파 = 추세 전환 |
| **보조 활용** | 이동평균선(MA)과 결합하여 ‘추세 확인’ + ‘변동성 확대 시점’ 포착 가능 |

📌 **한마디로:**

볼린저 밴드는 **“가격의 범위(Volatility Range)”를 측정하는 동적 지표**로,

**추세 + 변동성 + 과열도**를 한 번에 보여주는 **복합형 도구**입니다.

---

### **③ 실전 해석 (시그널 해석 및 활용법)**

| **해석 방식** | **의미** | **매매 해석** |
| --- | --- | --- |
| **가격이 상단 밴드 접촉** | 평균보다 2σ 이상 상승 → 과매수 또는 강한 상승 추세 지속 | 추세 중일 땐 “추세 지속 신호”, 횡보 중일 땐 “단기 과열 매도 주의” |
| **가격이 하단 밴드 접촉** | 평균보다 2σ 이상 하락 → 과매도 또는 약세 지속 | 하락추세 중일 땐 “추세 지속”, 횡보 중일 땐 “단기 반등 가능성” |
| **밴드 폭이 좁아짐 (수축)** | 변동성 감소 구간 → 에너지 축적 | 곧 큰 방향성 돌파(상승·하락) 발생 가능 |
| **밴드 폭이 확장됨 (확대)** | 변동성 급증 → 추세 진행 중 | 추세 방향으로 지속될 가능성 높음 |
| **가격이 밴드 밖으로 돌파 후 재진입** | ‘밴드 오버슈팅’ 발생 후 평균 복귀 | 단기 되돌림 가능성 (평균회귀 전략에 유리) |
| **밴드 중심선(MA20) 돌파** | 추세 전환 시그널 | 종가가 중심선을 상향 돌파 → 상승 전환, 하향 돌파 → 하락 전환 |

📊 **응용 패턴 예시**

- **볼린저 밴드 스퀴즈 (Bollinger Squeeze):** 밴드 폭이 극단적으로 좁아지면 곧 변동성 폭발 → 추세 돌파 신호
- **밴드 워크 (Band Walk):** 가격이 밴드 상단을 따라 이동 → 강한 추세 지속
- **밴드 리버전 (Band Reversion):** 밴드 바깥 돌파 후 다시 진입 → 평균 복귀 신호

### **④ 권장 기간 및 활용 타임프레임**

| **구분** | **권장 설정** | **주 사용 목적** | **신호 성격** |
| --- | --- | --- | --- |
| **단기 (스캘핑/데이)** | BB(10, 2.0) 또는 BB(12, 2.5) | 단기 변동성·급등락 포착 | 신속하지만 노이즈 많음 |
| **중기 (스윙)** | **BB(20, 2.0)** (기본값) | 추세 전환·밴드 수축 돌파 감지 | 안정적 표준형 |
| **장기 (포지션)** | BB(50, 2.0) 또는 BB(100, 2.0) | 대세 추세의 과열·침체 파악 | 신호 느리지만 정확도↑ |

📌 실전에서는 **BB(20,2)**가 표준이며,

EMA(20) 중심선으로 대체 시 **반응속도 향상** 가능.

---

### **⑤ 주의점 및 실전 팁**

1. **볼린저 밴드는 “과매수/과매도”를 절대 신호로 보지 말 것.**
    - 상단 터치 = 무조건 매도 ❌
    - 추세 강세 시 밴드 상단 ‘워킹(Band Walk)’ 현상 발생.
        
        → 상단 터치만으로는 매도 타이밍 아님.
        
2. **밴드 폭 수축(BB squeeze)은 “방향 예측”이 아닌 “폭발 예고”** 신호임.
    - 방향은 확인 불가 → 돌파 방향을 추세 필터(MACD, ADX 등)로 판별해야 함.
3. **볼륨(Volume)과 결합**하면 신뢰도 상승.
    - 밴드 돌파 + 거래량 급증 → 추세 전환 확률 ↑
4. **이평선 중심선(20MA)과 결합 매매 전략**
    - 종가 > 중심선 & 밴드 확장 → 상승 추세
    - 종가 < 중심선 & 밴드 확장 → 하락 추세
5. **레인지 장세에서는 평균회귀(Mean Reversion) 전략 유효.**
    - 밴드 상단 → 매도 / 하단 → 매수
    - 단, 반드시 변동성 수준(밴드 폭) 고려.

### **⑥ 종합 요약표 + 실전 응용 예시 + 최종 정리**

| **항목** | **내용** |
| --- | --- |
| **분류** | 복합형 지표 (변동성 + 추세 + 과열도) |
| **핵심 목적** | 변동성 측정 및 과열/침체 구간 식별 |
| **기준선** | 중심선(MA) = 추세 기준, 상·하단 밴드 = 변동성 경계 |
| **권장 설정** | 단기: (10,2.5) / 중기: (20,2.0) / 장기: (50,2.0) |
| **적합 시점** | 변동성 축적·폭발 구간, 추세 초기 단계 |
| **비적합 시점** | 변동성 낮고 거래량 부진한 종목 |
| **추천 조합** | BB + RSI (평균회귀 보조), BB + MACD/ADX (추세 확정), BB + Volume Profile (수급 확인) |

### **🔧 실전 응용 예시**

- **스윙 트레이딩용:**
    - BB(20,2) 기준
    - 밴드 폭 급수축 후 상단 돌파 + 거래량 급증 → 진입
    - 밴드 폭 확장 지속 + 중심선 이탈 시 익절 또는 청산
- **단기 매매용 (평균회귀):**
    - 하단 밴드 근접 + RSI<30 → 단기 매수
    - 상단 밴드 근접 + RSI>70 → 단기 매도
- **추세 매매용 (밴드 워크 확인):**
    - 종가가 상단 밴드 바깥에서 3봉 이상 지속 → 강세 지속 신호
    - 중심선 이탈 시 추세 약화 경고
