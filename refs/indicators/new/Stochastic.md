# Stochastic (스토캐스틱)

성격: 오실레이터
권장기간: 단기 (빠른 반응을 요구하는 트레이딩에 적합).
시그널: %K가 %D를 위에서 아래로 교차 → 매도 신호(과매수 상황에서 신뢰도↑). 다이버전스 활용.
권장 파라미터: 일반 14,3,3. 단기선(5,3,3) → 민감도↑.
정리된 블럭: 스토캐스틱(Stochastics) (https://www.notion.so/Stochastics-29a25237772c80f9aafcd613597e9630?pvs=21) 
설명: 현재 종가가 일정 기간(예: 14) 고저범위에서 어떤 위치에 있는지(%) 를 나타내는 오실레이터 (%K, %D).

단기 과매수·과매도 판단

## **📘 Stochastic Oscillator (스토캐스틱 오실레이터)**

> “현재 가격이 일정 기간 동안의 가격 범위에서 어느 위치에 있는가를 백분율로 표현한 모멘텀 오실레이터”
> 

스토캐스틱은 단순히 “현재 가격이 고가-저가 범위 내에서 얼마나 높은가”를 측정해,

**과매수·과매도 구간**과 **모멘텀 반전 시점**을 포착하는 지표입니다.

---

### **① 개념 및 계산식**

| **항목** | **내용** |
| --- | --- |
| **정의** | 일정 기간 동안의 **최고가·최저가 범위 대비 현재 종가의 상대적 위치**를 백분율로 나타낸 지표 |
| **기본 공식** | **%K = (현재 종가 - N일 최저가) / (N일 최고가 - N일 최저가) × 100** |
| **보조선 공식** | **%D = %K의 M일 이동평균** |
| **추가 보정(슬로우형)** | **Slow %K = %D(패스트 K의 평균)Slow %D = Slow %K의 이동평균** |
| **기본 설정** | %K 기간 = 14, %D 기간 = 3, Smoothing(평활화) = 3 |
| **핵심 의미** | 가격이 고점에 근접할수록 %K ↑, 저점에 근접할수록 %K ↓ |
| 즉, “최근 상승 모멘텀의 강약”을 반영함. |  |

---

### **② 지표 성격 (분류)**

| **항목** | **내용** |
| --- | --- |
| **분류** | **오실레이터형 (모멘텀 기반)** |
| **범위** | 0 ~ 100 (상·하한 고정형 오실레이터) |
| **핵심 목적** | 단기 과매수/과매도 감지 및 반전 타이밍 포착 |
| **주요 구성** | %K (빠른 반응선) / %D (신호선, %K의 평균선) |
| **활용 형태** | 빠른 %K와 느린 %D의 교차로 매매 타이밍 판단 |
| **장점** | 단기 변동에 민감 — 반전 포착 빠름 |
| **단점** | 강한 추세장에서는 과매수·과매도 신호 무력화 |

---

### **③ 실전 해석 (시그널 및 활용법)**

| **해석 구간** | **의미** | **실전 매매 포인트** |
| --- | --- | --- |
| **%K > 80** | 과매수 구간 — 단기 상승 과열 | 단기 조정 또는 익절 구간 (단, 추세강세 시 지속 가능) |
| **%K < 20** | 과매도 구간 — 단기 하락 과도 | 단기 반등 가능성, 매수 관심 구간 |
| **%K가 %D 상향 돌파** | 강세 전환 신호 (골든 크로스) | 매수 신호 |
| **%K가 %D 하향 돌파** | 약세 전환 신호 (데드 크로스) | 매도 신호 |
| **다이버전스** | 가격과 스토캐스틱의 방향이 반대 | 추세 전환 가능성 (특히 단기봉에서 강력) |

> ⚠️
> 
> 
> **실전 포인트:**
> 

> 단순히 80 이상이면 매도, 20 이하면 매수가 아니라
> 

> **%K가 하락·상승 전환하는 ‘방향성 변화 시점’**에 주목해야 합니다.
> 

---

### **④ 권장 기간 및 타임프레임**

| **구분** | **권장 설정** | **특징 및 목적** |
| --- | --- | --- |
| **단기 (5~9)** | %K 5~9, %D 3 | 빠른 반응, 스캘핑·데이 트레이딩용 |
| **중기 (14)** | %K 14, %D 3 | 표준 설정, 스윙 및 일반 추세 분석용 |
| **장기 (20~21)** | %K 2021, %D 35 | 노이즈 감소, 추세 지속성 판단용 |

> 💡
> 
> 
> **조합 예시:**
> 
- Stoch(14,3,3) + ADX(14): 추세 강도와 반전 타이밍 동시 체크
- Stoch(9,3,3) + EMA(20): 단기 눌림목 진입용
- Stoch RSI: RSI 변동성 반영형 (심화형 오실레이터)

---

### **⑤ 주의점 및 실전 팁**

| **항목** | **설명** |
| --- | --- |
| **1. 강한 추세장에서는 역효과** | 상승추세 중엔 과매수 구간이 오래 지속될 수 있음 — 추세 필터(EMA/ADX) 병행 필수 |
| **2. 단독 사용 위험** | 횡보장에서는 효과적이지만, 방향성 확인 없이 쓰면 잦은 가짜 신호 발생 |
| **3. Slow Stoch 권장** | Fast형(9,3)은 민감하지만 노이즈 많음 → Slow형(14,3,3)이 실전에서는 안정적 |
| **4. 다이버전스 신호 중시** | 과매수·과매도보다 가격-지표 괴리가 실제 반전 신호로 더 신뢰도 높음 |
| **5. 타 지표와 조합** | RSI, MACD, ADX와 병행 시 신뢰도 극대화 가능 |

### **⑥ 종합 요약표**

| **항목** | **내용** |
| --- | --- |
| **분류** | 오실레이터형 (모멘텀 지표) |
| **범위** | 0 ~ 100 |
| **핵심 구성** | %K (빠른선), %D (신호선) |
| **표준 설정** | (14, 3, 3) |

| **권장 기간** | 단기 59 / 중기 14 / 장기 2021 |
| **추천 조합** | Stoch + ADX / Stoch + EMA / Stoch + RSI |
| **적합 시장** | 횡보 또는 단기 반등 구간 |
| **비적합 시장** | 강한 추세장 (신호 왜곡 위험) |

### **실전 응용 예시**

- **단기 트레이딩용**
    - Stoch(9,3,3)
    - %K<20 → 상향 돌파 → 매수
    - %K>80 → 하향 전환 → 매도
- **스윙 트레이드용**
    - Stoch(14,3,3)
    - RSI 50 이상 + Stoch 골든크로스 = 상승 지속 신호
    - ADX 20 미만 + Stoch 데드크로스 = 횡보 반락 신호
