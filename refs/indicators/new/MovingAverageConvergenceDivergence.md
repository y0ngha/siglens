# MACD (Moving Average Convergence Divergence)

성격: 오실레이터, 추세 전환 지표, 추세 지표
권장기간: 중기(스윙) 중심.
시그널: - MACD 선과 시그널선 교차, 히스토그램 수평 전환(0선 교차 포함)

- 다이버전스.
권장 파라미터: - 단기: 5, 20
- 중기: 5, 40
- 장기: 20, 40
- 평균: 일봉 12/26/9 (중기, 스윙)
주의: 추세가 강할 때 교차 신호는 지연될 수 있음 — ADX 등으로 추세 강도 확인.
정리된 블럭: 제목 없음 (https://www.notion.so/28b25237772c8182913bef34014c0423?pvs=21) 
설명: 단기 EMA(보통12) − 장기 EMA(26) = MACD 선, Signal(9)과의 교차/히스토그램(차이)으로 표현.

## **📘MACD (Moving Average Convergence Divergence)**

> “이동평균선의 수렴·확산을 통해 추세 전환과 모멘텀 강도를 포착하는 지표”
> 

> 가격 자체보다
> 
> 
> **단기·중기 EMA 간 차이**
> 

---

### **① 개념 및 계산식**

| **항목** | **내용** |
| --- | --- |
| **정의** | MACD는 **단기 EMA와 장기 EMA의 차이를 기반**으로 한 모멘텀 지표입니다. 추세 방향뿐 아니라, 추세 강도와 전환 가능성까지 포착 가능. |
| **공식** | 1. MACD Line = EMA(단기) − EMA(장기) 2. Signal Line = EMA(MACD Line, 시그널기간) 3. Histogram = MACD Line − Signal Line |
| **표준값** | EMA 단기 12 / EMA 장기 26 / Signal 9 (일봉 기준) |
| **핵심 의미** | - MACD Line > Signal → 단기 모멘텀 강세 - MACD Line < Signal → 단기 모멘텀 약세 - Histogram 크기 → 모멘텀 강도/약화 표시 |

> 💡
> 
> 
> **한마디:**
> 

> MACD는 “단기·중기 EMA의 상호작용”으로
> 
> 
> **추세 전환 신호와 모멘텀 강약**
> 

---

### **② 지표 성격 (분류)**

| **항목** | **내용** |
| --- | --- |
| **분류** | 모멘텀 + 추세형 혼합 지표 (Hybrid Indicator) |
| **범위** | 무제한형 (±∞) — 수치 자체 제한 없음 |
| **주요 목적** | 추세 전환 포착, 모멘텀 강약 확인 |
| **장점** | 단순 이동평균보다 **추세 전환 선행 가능**, Histogram으로 시각적 강도 확인 |
| **단점** | 횡보장에서는 신호 노이즈 발생 가능 |

---

### **③ 실전 해석 (시그널 및 활용법)**

| **시그널** | **의미** | **실전 매매 해석** |
| --- | --- | --- |
| **MACD Line 교차 Signal Line 상향** | 단기 모멘텀 상승 전환 | 매수 신호 (Golden Cross) |
| **MACD Line 교차 Signal Line 하향** | 단기 모멘텀 하락 전환 | 매도 신호 (Death Cross) |
| **Histogram 상승 → MACD 확대** | 모멘텀 강세 강화 | 추세 지속, 눌림목 매수 유리 |
| **Histogram 하락 → MACD 축소** | 모멘텀 약화 | 추세 전환 가능성, 익절/손절 판단 |
| **0선 교차** | MACD Line이 0 위 → 단기 EMA > 장기 EMA (강세) 0 아래 → 단기 EMA < 장기 EMA (약세) | 추세 방향 확인 기준선 |
| **다이버전스** | 가격과 MACD 방향 불일치 | 추세 약화/전환 가능성 신뢰도 높음 |

> 📊 **예시:**
> 

> TSLA가 최근 고점에서 MACD Histogram 하락 + MACD Line Signal Line 하향 교차 → 단기 매도 신호로 활용.
>

### **④ 권장 기간 및 활용 타임프레임**

| **구분** | **권장 기간** | **활용 목적** |
| --- | --- | --- |
| **단기 (분봉/1시간봉)** | EMA(6,12) / Signal 3~5 | 데이 트레이딩, 단기 진입 타이밍 |
| **중기 (일봉/4시간봉)** | EMA(12,26) / Signal 9 | 스윙 트레이딩, 추세 전환 포착 |
| **장기 (주봉/월봉)** | EMA(20,5060) / Signal 912 | 중장기 추세 확인, 포지션 트레이딩 |

> 
>

### **⑤ 주의사항 및 실전 팁**

| **항목** | **내용** |
| --- | --- |
| **주의1** | 횡보 구간에서는 Signal 교차 신호 빈번 → 가짜 신호 주의 |
| **주의2** | 지표 수치보다 **Histogram 방향과 크기**로 모멘텀 강도 판단 |
| **주의3** | 다이버전스 시 신뢰도 ↑ (가격 고점/저점과 MACD 불일치 확인) |
| **활용 팁1** | MACD + EMA 20~60 → 추세 확인 필터 역할 |
| **활용 팁2** | MACD + RSI/Stochastic → 단기 과매수/과매도 보조 |
| **활용 팁3** | MACD 0선 교차 → 장기 추세 전환 확인용 |

### **⑥ 종합 요약표**

| **항목** | **내용** |
| --- | --- |
| **분류** | 모멘텀 + 추세형 혼합 지표 (Hybrid) |
| **주요 목적** | 추세 전환 포착 및 모멘텀 강도 확인 |
