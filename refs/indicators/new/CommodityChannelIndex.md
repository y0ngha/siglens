# CCI (Commodity Channel Index, 상품채널지수)

성격: 오실레이터
정리된 블럭: CCI(Commodity Channel Index) (https://www.notion.so/CCI-Commodity-Channel-Index-29a25237772c80fb83c7cfe4596089b2?pvs=21) 
설명: 평균 이탈도 기반으로 시장의 과열·침체 국면 포착

---

## **📘 CCI (Commodity Channel Index)**

| **항목** | **내용** |
| --- | --- |
| **한글명** | 상품채널지수 |
| **영문명** | Commodity Channel Index |
| **개발자** | Donald Lambert (1980년) |
| **주요 목적** | 가격이 통계적으로 평균가로부터 얼마나 벗어났는지를 측정 (평균 이격률 기반의 과열·과매도 판단) |
| **지표 유형** | 오실레이터형 (Oscillator) — 변동성/이격도 기반 |

---

### **① 개념 및 수식 정의**

**개념**

CCI는 “현재 가격이 일정 기간 평균 가격(평균 이동가)으로부터 얼마나 떨어져 있는가”를 측정합니다.

즉, **가격의 통계적 편차(Deviation)** 를 이용하여 과매수·과매도 구간을 정량화하는 지표입니다.

RSI처럼 일정 범위(0~100)에 제한되지 않고, ±300까지 확장될 수 있는 **비한정형 오실레이터**입니다.

---

**기본 수식**

CCI = \frac{(TP - SMA(TP, n))}{0.015 \times MD}

| **기호** | **의미** |
| --- | --- |
| TP | Typical Price = (고가 + 저가 + 종가) / 3 |
| SMA(TP, n) | n기간 단순이동평균 (기준가격) |
| MD | 평균편차 = 평균( |
| 0.015 | 표준화 상수 (데이터 분포를 일정 범위로 맞추기 위한 조정값) |

→ **핵심 해석:**

- CCI 값이 **+100 이상** → 가격이 평균보다 “너무 높음” (과열 or 강한 상승)
- CCI 값이 **100 이하** → 가격이 평균보다 “너무 낮음” (과매도 or 강한 하락)

---

### **② 지표 성격 (분류 및 특성)**

| **구분** | **분류** | **설명** |
| --- | --- | --- |
| **지표 유형** | 오실레이터형 (Oscillator) | 평균 이격도를 계산하여 과열·과매도 구간 파악 |
| **핵심 성격** | 추세선 돌파 이전에 과열 신호를 먼저 제공 (선행형 지표 성격) |  |
| **민감도 조절** | 기간(n)이 짧을수록 반응 빠름 (단타용), 길수록 완화됨 (스윙용) |  |
| **한계** | 횡보 구간에서는 신호 과다 발생 가능 |  |

---

### **③ 실전 해석 (시그널 해석 및 활용법)**

| **구분** | **조건** | **해석 및 매매 전략** |
| --- | --- | --- |
| **매수 신호 (Buy)** | CCI가 -100 아래 → 상향 돌파 | 과매도 구간에서 반등 신호 |
| **매도 신호 (Sell)** | CCI가 +100 위 → 하향 돌파 | 과매수 구간에서 조정 신호 |
| **추세 지속 신호** | CCI가 +100 이상에서 유지 | 강한 상승추세 지속 중 (추가 매수 고려) |
| **추세 약화 신호** | CCI가 고점 하락 (Lower High) | 상승 피로 누적, 하락 반전 가능성 |
| **중앙선(0선) 돌파** | CCI가 0 위/아래 돌파 | 추세 전환의 초기 신호 (모멘텀 방향 확인용) |

> 🔸
> 
> 
> **CCI는 단순히 “과매수·과매도”로 끊기보단,**
> 

> “+100 이상 구간 지속” → 상승 추세 강화,
> 

> “-100 이하 지속” → 하락 추세 강화로 해석하는 것이
> 
> 
> **현대적 트레이딩 방식**
>

### **④ 권장 기간 및 타임프레임**

| **구분** | **권장 세팅** | **특징 및 활용 목적** |
| --- | --- | --- |
| **단기 (스캘핑/단타)** | 10~14 | 빠른 반응성, 변동성 장세 대응 |
| **중기 (스윙)** | 20 | 표준 세팅 (기본값 20), 가장 보편적 |
| **장기 (추세추종)** | 30~50 | 노이즈 제거, 큰 파동 중심 확인 |

> 📊
> 
> 
> **추천 타임프레임:**
> 
- 단타 → 5분, 15분봉
- 스윙 → 1시간, 4시간봉
- 중장기 → 일봉, 주봉

---

### **⑤ 주의점 및 실전 팁**

| **주의점** | **설명** |
| --- | --- |
| **횡보장에서 잦은 신호 발생** | 노이즈가 많을 때, +100/-100 돌파 신호가 반복되어 신뢰도 저하 |
| **추세 강도 확인 필요** | ADX, MACD 등과 병행 시 “가짜 반전” 걸러낼 수 있음 |
| **변동성 장세에서는 지연 반응** | 급등락 시 평균편차가 커져 오히려 반응이 늦어짐 |
| **+100/-100 구간 지속에 주목** | 단순 돌파보다, 해당 구간 “유지 시간”을 추세 강도로 판단해야 함 |

> 💡
> 
> 
> **실전 팁:**
> 

> RSI, Stochastic 등과 함께 사용하면
> 

> “모멘텀의 방향 + 과열 강도”를 동시에 측정 가능함.
> 

> 예: RSI 상승 중, CCI가 +100 돌파 → 강한 매수세 지속 신호.
>

### **⑥ 종합 요약표 + 실전 예시**

| **구분** | **내용** |
| --- | --- |
| **지표 유형** | 오실레이터형 (변동성·이격도 기반) |
| **핵심 목적** | 평균 가격 대비 괴리도(Deviation) 측정 |
| **주요 신호** | ±100 돌파 및 0선 교차 |
| **표준 세팅** | 기간 20 |
| **해석 기준** | +100 이상 = 과열, -100 이하 = 과매도 |
| **강점** | 추세 초기 감지, 변동성 시장에서 선행성 있음 |
| **약점** | 횡보 시 잦은 반전 신호 |
| **추천 조합** | RSI, MACD, ADX, MA (추세 확인용) |

---

### **💬 실전 응용 예시**

| **상황** | **해석 및 전략** |
| --- | --- |
| CCI가 -150 → -100 상향 돌파 | 과매도 해소 + 반등 시그널 → 매수 진입 고려 |
| CCI가 +200 이상 유지 | 강한 상승추세 지속 (단기 익절 고려) |
| CCI가 +100 돌파 후 하락하며 0선 돌파 | 추세 전환 가능성 → 매도 대응 |
| CCI와 가격의 **다이버전스 발생** | 반전 가능성 매우 높음 (특히 장기봉에서 유효) |

---

### **✅ 최종 정리**

> CCI = “가격의 평균 이격률로 보는 과열 지표”
±100 단순 돌파보다, **그 구간의 지속성**과
**가격 다이버전스**를 함께 분석하는 것이 핵심입니다.
RSI보다 변동성에 민감하고, 추세형 신호와 조합 시 정확도가 상승합니다.
스윙 트레이더에게는 “추세 전환 초기 감지용”으로 가장 유용한 오실레이터 중 하나입니다.
